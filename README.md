# Методика исследования ИИ-изображений

Публичный рабочий репозиторий предвалидационной методики судебно-экспертного исследования цифровых изображений на признаки создания или изменения с использованием систем искусственного интеллекта.

Владелец проекта: **Смагулов Амирхан Каиржанович**.

## Статус

Проект находится на стадии открытого dry run и подготовки к валидации. Он не включён в Государственный реестр методик судебно-экспертных исследований Республики Казахстан и не предназначен для категорических экспертных выводов до завершения установленной процедуры апробации и валидации.

## Что уже работает

- локальная CUDA-среда на NVIDIA GeForce RTX 5050;
- Community Forensics 384 и SPAI;
- c2patool, PyTorch и Docker с GPU;
- SOP-M0 по приёму, сохранению и пригодности объекта;
- реестр датасета, SHA-256 и parent-child связи оригиналов и производных;
- первый dry run на 20 документированных реальных фотографиях;
- автоматизированный импорт HEIC с контролируемым преобразованием в JPEG.

На первом real-only dry run Community Forensics дал один, а SPAI шесть условных ложных сигналов не ниже 0.5. Совпавших сигналов не было. Это подтверждает, что score и порог 0.5 нельзя использовать как самостоятельный экспертный вывод.

## Данные и приватность

Личные фотографии, веса моделей, скачанные репозитории детекторов, виртуальные окружения и рабочие outputs не публикуются. В репозитории хранятся только обезличенные результаты, документация и воспроизводимые скрипты.

Крупные сторонние PDF, DOCX и PPTX также не публикуются в этом публичном репозитории. Они сохраняются локально и в частном корпусе проекта; в Git остаются их описания и ссылки.
