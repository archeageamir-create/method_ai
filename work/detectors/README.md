# Локальные AI-детекторы

Эта папка содержит локальные рабочие копии исследовательских AI-детекторов.

- `community-forensics` — основной кандидат; рабочий профиль использует модель 384.
- `universal-fake-detect` — исторический контроль; не рабочий кандидат.
- `spai` — второй рабочий кандидат с независимым спектральным подходом.
- `clipbased` — проверенный, но не выбранный кандидат.
- `.venv-community` и `.venv-ufd` — раздельные Python-окружения.
- `models` — локальные веса; в Google Drive не синхронизируются из-за размера.

Для SPAI на RTX 5050 8 GB рабочий wrapper по умолчанию ограничивает длинную сторону до 1024 px и использует два data workers. Полноразмерные фотографии iPhone 4284×5712 с восемью workers переполняют VRAM общей GPU-памятью и неоправданно замедляют inference. Параметры можно изменить через `-ResizeTo` и `-DataWorkers`, но они должны быть записаны в журнал запуска.

Результаты детекторов не являются самостоятельным экспертным выводом. До отдельной валидации они используются только в исследовательском режиме.
