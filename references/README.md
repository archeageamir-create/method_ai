# Локальный корпус источников

Эта папка содержит локальные контролируемые копии материалов, ранее находившихся только в Google Drive:

- `normative/rk/` — снимки нормативных страниц Республики Казахстан;
- `scientific/general/` — общие руководства NIST и SWGDE;
- `scientific/detectors/` — статьи по детекторам и обобщаемости;
- `scientific/validation/` — материалы по оценке дискриминаторов;
- `scientific/provenance/` — снимок спецификации C2PA;
- `presentations/` — внешний референс презентации;
- `reviews/` — критическая рецензия проекта.

Крупные PDF, DOCX и PPTX из этого корпуса хранятся локально и в частном Google Drive, но исключены из публичного GitHub. В Git публикуются только безопасные текстовые снимки, описания, ссылки и собственная документация проекта.
