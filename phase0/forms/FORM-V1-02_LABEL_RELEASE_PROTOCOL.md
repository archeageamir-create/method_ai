## Проверка перед раскрытием

- [ ] получены raw outputs всех заявленных модулей;
- [ ] количество outputs сопоставлено с input manifest;
- [ ] SHA-256 raw outputs зафиксированы;
- [ ] action log закрыт для редактирования;
- [ ] технические отказы перечислены без знания labels;
- [ ] повторные запуски перечислены и соответствуют preregistration;
- [ ] operating point не изменялся;
- [ ] код, веса и preprocessing не изменялись;
- [ ] исполнитель письменно подтвердил отсутствие доступа к labels;
- [ ] держатель labels подтвердил целостность sealed manifest.

## Отклонения

Если выявлена утечка, потеря output, неразрешённый повторный запуск или смена версии, labels не раскрываются до решения о действительности теста.

>

## Решение

- [ ] labels разрешено раскрыть;
- [ ] раскрытие остановлено, тест требует расследования;
- [ ] тест признан недействительным.
