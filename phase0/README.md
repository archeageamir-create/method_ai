# Стартовый пакет проекта

Дата создания: 13 июля 2026 года. Актуализировано 14 июля 2026 года.

Этот каталог содержит документы этапа 0 для проекта методики судебно-экспертного исследования ИИ-изображений.

Текущий владелец и единственный участник проекта: **Смагулов Амирхан Каиржанович**.

Принятый первый пилот по специальности 4.1: JPEG/PNG, основная задача — различение обычного фотографического изображения и изображения, полностью созданного системой ИИ.

Файлы:

- `PROJECT_CHARTER.md` — паспорт проекта;
- `PILOT_SCOPE_V1.md` — границы первого пилота;
- `DECISION_POLICY_V1.md` — единые правила категорий H0/Ht/Hg/Hu и обязательного отказа от решения;
- `sop/SOP-M0_INTAKE_PRESERVATION_SUITABILITY.md` — приём, оригинал, SHA-256, рабочая копия и пригодность;
- `forms/` — карточка объекта, цепочка передачи и журнал действий;
- `RACI.md` — текущие роли;
- `MASTER_PLAN.md` — последовательность работ и контрольные ворота;
- `DECISION_LOG.md` — журнал решений;
- `RISK_REGISTER.md` — реестр рисков;
- `CHANGELOG.md` — история изменений;
- `DOCUMENT_REGISTER.md` — реестр документов;
- `ISSUE_REGISTER.md` — открытые задачи;
- `TECHNICAL_ENVIRONMENT_BASELINE.md` — подтверждённая конфигурация компьютера и инструментов;
- `requirements-pytorch-cu132.txt` — зафиксированные зависимости проверенного PyTorch-окружения;
- `REVIEW_RESPONSE_MATRIX.md` — статус замечаний критической рецензии.
