# Словарь значений датасета V0.1

## `ground_truth`

- `real` — исходное изображение получено камерой или из проверенного реального датасета;
- `ai_generated` — изображение сгенерировано известным сервисом/моделью в контролируемых условиях;
- неизвестное происхождение в валидационный набор не включается.

## `truth_basis`

- `controlled_capture` — собственная контролируемая фотосъёмка;
- `controlled_generation` — собственная генерация с сохранённой записью модели и параметров;
- `trusted_dataset` — проверенный открытый датасет с документированным происхождением;
- `provided_with_provenance` — файл передан с проверяемой историей происхождения.

## `transformation`

- `none` — оригинал без изменений;
- `jpeg_reencode` — повторное JPEG-сохранение;
- `resize` — изменение размера;
- `crop` — обрезка;
- `webp_reencode` — сохранение в WebP;
- `screenshot` — контролируемый скриншот;
- `blur` — контролируемое размытие;
- `combined` — заранее описанная комбинация.

## `split`

- `development` — можно видеть labels и результаты;
- `internal_holdout` — labels скрыты до заморозки процедуры;
- `external_holdout` — независимый финальный набор.
