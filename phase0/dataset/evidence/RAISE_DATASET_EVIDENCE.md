# Evidence card — RAISE

| Поле | Зафиксированное значение |
|---|---|
| Dataset | RAISE — The Raw Images Dataset |
| Official site | https://loki.disi.unitn.it/RAISE/ |
| Reported size | 8156 high-resolution RAW images |
| Acquisition | 4 photographers, 3 cameras, 2011–2014, more than 80 places |
| Integrity claim | camera-native, never processed |
| Required citation | Dang-Nguyen et al., RAISE, ACM MMSys 2015 |
| Evidence checked | 2026-07-14 |

Авторы указывают, что набор создан для digital image forensics и сформирован без типичных проблем copyright/privacy web-коллекций.

Статус: `candidate`. На официальной странице не зафиксирован стандартный SPDX/Creative Commons identifier; до скачивания архивируются фактические access/download terms. Для детекторов нужен отдельный замороженный RAW→JPEG/PNG conversion profile.
